Selecting Elements in the DOM** To manipulate elements, you first need to **select** them. ### Methods to Select Elements | Method | Returns | Example | |--------|---------|---------| | `document.getElementById("id")` | Single element (by ID) | `document.getElementById("title")` | | `document.querySelector("selector")` | First matching element | `document.querySelector("#title")` | | `document.querySelectorAll("selector")` | All matching elements (NodeList) | `document.querySelectorAll("p")` | | `document.getElementsByClassName("class")` | HTMLCollection | `document.getElementsByClassName("btn")` | | `document.getElementsByTagName("tag")` | HTMLCollection | `document.getElementsByTagName("h1")` | ### Example ```js const heading = document.getElementById("title"); console.log(heading.textContent); // Output: Hello, DOM! ``` --- **# 3. Changing Elements in the DOM** Once selected, you can modify elements. ###Modify Text ```js document.getElementById("title").textContent = "Welcome to JavaScript!"; ``` ### Modify HTML ```js document.getElementById("title").innerHTML = "Updated Title"; ``` ### Change Styles ```js document.getElementById("title").style.color = "blue"; document.getElementById("title").style.fontSize = "24px"; ``` --- **# 4. Handling Events** You can **respond to user interactions** (e.g., clicks, typing, hovering). ### Adding an Event Listener ```js document.getElementById("btn").addEventListener("click", function() { alert("Button Clicked!"); }); ``` ### Example: Change Text on Click ```js document.getElementById("btn").addEventListener("click", function() { document.getElementById("title").textContent = "Button Clicked!"; }); ``` --- **# 5.
